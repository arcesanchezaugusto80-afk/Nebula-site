Nebula - Sitio web (demo)
Archivos incluidos:
- index.html
- styles.css
- script.js
- README.txt

Instrucciones:
1) Descomprimí nebula_site.zip.
2) Abrí index.html en tu navegador para ver el sitio localmente.
3) Reemplazá textos, precios, imágenes y datos de contacto por los reales.
4) Para subirlo online: podés usar GitHub Pages, Netlify o Vercel (son gratuitos y fáciles).

Notas:
- Las imágenes son placeholders. Sustituí las secciones .image-placeholder/.photo con <img src="tu-imagen.jpg"> si querés.
- Si querés que lo deje con fotos reales y un formulario que mande correos, decímelo y lo preparo.
